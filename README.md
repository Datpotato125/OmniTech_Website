# OmniTech Website

A fully static front-end website for **OmniTech Corp** — a fictional technology company based in Exton, Pennsylvania. The site showcases a unified ecosystem of personal devices, smart home appliances, immersive computing hardware, and a cloud/AI platform.

```
└── css/
    ├── style.css          # Global styles, variables, nav, footer
    └── product.css        # Product page layout (hero, specs, features, variants)
```

No build tools, no frameworks, no dependencies. Every page is plain HTML/CSS with vanilla JavaScript where needed.

---

## Tech Stack

- **HTML5** — semantic markup throughout
- **CSS3** — custom properties (variables), CSS Grid, Flexbox, `@media` queries
- **Vanilla JavaScript** — account dashboard, session simulation, dynamic panel rendering
- **No frameworks** — zero npm, zero bundlers, zero external libraries

---

## Account Dashboard

`account.html` is a full single-page dashboard with:

- **Sidebar navigation** with fixed positioning
- **User panels** — Overview, Devices, Storage, Payments, Security, Activity Log, Profile Settings
- **Admin panels** — Admin Overview, User Management, Product Catalog, Reviews & Moderation, Financial Dashboard, System Health
- **Session simulation** — user data stored in `localStorage`; admin access unlocked via a hardcoded admin email check
- **Dynamic rendering** — all panel content generated by JavaScript from in-page data structures (`USERS_DB`, `ACTIVITY_LOG`, etc.)

## Design System

All visual tokens are defined as CSS custom properties in `style.css`:

```css
--blue:    #0066ff
--cyan:    #00ffff
--dark:    #0a0a0a
--surface: rgba(255,255,255,0.03)
--border:  rgba(255,255,255,0.08)
--white:   #f0f0f0
--mono:    'Courier New', monospace
```

The design language is dark, high-contrast, and terminal-inspired — monospace labels, cyan accents, subtle borders, and minimal decoration.
