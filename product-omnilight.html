Key Features

Light That Adapts to Life.

Universal

64 Devices — Any Brand

OmniLight Hub speaks Zigbee, Z-Wave, Thread, Matter, and Bluetooth Mesh simultaneously. It controls Philips Hue, IKEA TRADFRI, Sengled, LIFX, and hundreds of other brands alongside OmniTech's own bulbs.

Circadian

Science-Based Light Schedules

OmniAI mirrors the sun's color temperature arc — cool blue-white (6500K) during work hours to support alertness, warming to amber (2700K) in the evening to signal the body to produce melatonin.

Presence

Follow-Me Room Automation

Using OmniTherm room sensors and OmniPhone location, OmniLight turns on rooms as you enter and off as you leave — no app interaction, no motion sensor lag, no wasted electricity.

Security

Alert-Triggered Flood Lighting

When OmniSecure detects motion at night, OmniLight floods connected outdoor fixtures at full brightness for 60 seconds and sends a push notification — deterrence and documentation, coordinated.

Local

Zero Cloud Dependency

All schedules, scenes, and automations run locally on the hub processor. Internet outages, server downtime, and app unavailability have zero impact on lighting operation in your home.

OmniDev

Open API for Custom Integrations

OmniLight Hub exposes a REST and WebSocket API for developers — build custom lighting apps, integrate with home theater systems, or create biometric-triggered scenes via OmniWatch data.
