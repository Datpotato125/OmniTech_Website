Key Features

Maximum Everything.

OmniChip M2 Max

400 GB/s Memory Bandwidth

The M2 Max's 400 GB/s unified memory bandwidth is faster than any discrete GPU memory system at this power envelope. 3D renders, 8K timeline editing, and large model inference run in real time.

Display

2,000-Nit XDR OLED

OmniBook Studio's ProMotion XDR display sustains 1,600 nits — the threshold for true HDR mastering — making it the only laptop qualified for professional HDR color grading without an external reference monitor.

Storage

Up to 4 TB at 7.4 GB/s

4 TB of NVMe Gen 4 storage reads at 7.4 GB/s — faster than most external RAID arrays. 8K ProRes RAW timelines play back directly from internal storage without proxies.

Media Engine

Hardware ProRes & AV1

A dedicated 16-core Media Engine handles ProRes, H.265, H.264, and AV1 encode and decode in hardware. Export a 4K ProRes file in a fraction of the time of software rendering.

Multi-Display

Drive 4 External 6K Displays

Connect up to four 6K Pro Display XDR monitors simultaneously via Thunderbolt 4 — while the built-in display stays on. Total pixel count exceeds a 100-inch 8K screen.

Battery

22 Hours — 100 Wh

100 Wh — the maximum allowed on commercial aircraft — delivers up to 22 hours. M2 Max's efficiency means full-load GPU rendering still leaves hours of charge for the rest of your day.
